## Инструкция по запуску

1. Выполните команду в терминале:
    ```bash
    npm install
    ```

2. Создайте файл `.env`, скопировав содержимое из `.env.example`:
    ```bash
    cp .env.example .env
    ```

3. Выберите подходящую команду для запуска проекта в режиме разработки:
    ```bash
    npm run dev
    # или
    yarn dev
    # или
    pnpm dev
    # или
    bun dev
    ```
